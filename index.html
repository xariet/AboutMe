Pagination in Tableau

In this post I will share how to set table pagination in Tableau.

  • Create a large table using Superstore dataset and create a parameter “Page size” which indicates the number of results you would like to see.
  • Create a calculated field “Index” which will return the index of the current row in partition, without any sorting with regard to value. Convert it to discrete and bring to rows.
  • Create page number calculation field which indicates the number of page that you would like to see. Convert it to discrete and bring in to filter shelf.
